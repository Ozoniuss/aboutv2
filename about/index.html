My name is Alex. Online, I go by the name of Ozoniuss. The story of this name comes from back in the days when I was doing math contests. There used to be a running joke that if we didn't know how to justify a part of a solution, we'd write something like "from Ozoniuss's theorem, we can deduce [that part]" hoping that the dude grading it would fall for it. Yeah I know, 13 year olds. But imagine if someone actually fell for that: it would've been fucking hilarious. Probably the best math contest story ever.
